## 🚀 Overview
**SplitSmart Pro** is a modern, high-performance web application designed to simplify shared finances for friends, roommates, and travel groups. Built with a focus on speed, aesthetics, and algorithmic efficiency, it helps you track expenses and find the fastest way to settle up.

## ✨ Key Features
- 📊 **Intelligent Expense Tracking**: Log expenses with detailed descriptions, automated splitting, and support for both individual and group-based payers.
- 🧠 **Optimized Settlements**: Powered by a "Greedy Algorithm" that minimizes the total number of transactions required to balance the books.
- 👥 **Group Management**: Create custom groups, manage members, and see collective balances at a glance.
- 💎 **Premium UI/UX**: A sleek, dark-mode-ready interface featuring glassmorphism, smooth micro-animations, and a responsive layout for mobile and desktop.
- ✅ **One-Tap Payments**: Mark individual transactions or entire settlements as "Paid" with intuitive controls.
- 🛡️ **Safe Deletions**: Built-in confirmation workflows to prevent accidental record loss.

## 🛠️ Technology Stack
- **Frontend**: [React 19](https://react.dev/) with [Hooks](https://react.dev/reference/react)
- **Language**: [TypeScript](https://www.typescriptlang.org/) for robust typing
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for modern, utility-first design
- **Icons**: [FontAwesome 6](https://fontawesome.com/)
- **Build Tool**: [Vite](https://vitejs.dev/) for lightning-fast development

## 🏃 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- npm or yarn

### Local Installation
1. **Clone and Enter**:
   ```bash
   # Navigate to the project directory
   cd splitsmart-pro
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Launch Development Server**:
   ```bash
   npm run dev
   ```
   *Your app will be available at `http://localhost:5173`*
